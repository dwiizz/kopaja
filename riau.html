Cara mengetahui pajak kendaraan untuk wilayah Provinsi Riau yaitu sebagai berikut :

  • Isi form Plat Nomor kendaraan menggunakan spasi. Contoh BM 1234 ABC.
  • Isi 5 digit nomor rangka terakhir. Silakan lihat di STNKnya.

Setelah itu silakan klik tombol PAJAK untuk cek pajak atau BBN untuk cek bisaya pokok baliknama. Jika kendaraan anda terdaftar di dalam database SAMSAT, maka data kendaraan anda akan muncul.
Jika data kendaraan anda tidak muncul, bisa jadi kendaraan anda telah diblokir. Silakan kunjungi Samsat Terkait untuk info lengkapnya.

Silakan klik tombol berikut untuk mulai cek pajaknya KLIK DISINI
